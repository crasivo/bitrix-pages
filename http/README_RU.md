HTTP ресурсы
===

В данной директории предполагается хранить дополнительные ресурсы для отладки сетевого слоя.

Список файлов и их назначение:

- [rest_v1_openapi.yaml](rest_v1_openapi.yaml) — спецификация OpenAPI для REST контроллеров _(Crasivo\Pages\Rest)_
- [rest_v1_requests.http](rest_v1_requests.http) — набор готовых Request запросов для выполнения через HTTP клиент ([PhpStorm](https://www.jetbrains.com/help/phpstorm/http-client-in-product-code-editor.html)).
